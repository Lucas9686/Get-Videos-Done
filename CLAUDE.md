# Get Videos Done - VideoFlow Project

This is a VideoFlow video production project powered by kie.ai + ElevenLabs + Remotion.

## Workflow

Use the `/videoflow:*` slash commands to produce videos:

1. `/videoflow:new-video` — Describe your video idea, get a brief + scene plan
2. `/videoflow:plan-scenes` — Plan which AI models and prompts for each scene
3. `/videoflow:generate-assets` — Generate images & video clips via kie.ai
4. `/videoflow:generate-audio` — Generate voiceovers & SFX via ElevenLabs
5. `/videoflow:compose` — Build Remotion composition from all assets
6. `/videoflow:render` — Render final MP4
7. `/videoflow:review` — Review quality against brief
8. `/videoflow:progress` — Check status

## API Keys Required

- `KIE_API_KEY` — kie.ai API key (https://kie.ai/api-key)

## Project Structure

- `src/` — Remotion React components
- `src/videoData.ts` — Scene data (generated by /videoflow:compose)
- `src/components/` — Scene, Transition components
- `assets/images/` — AI-generated images
- `assets/clips/` — AI-generated video clips
- `assets/audio/` — Voiceovers, sound effects, music
- `.videoflow/` — Project state (BRIEF.md, SCENES.md, ASSET-PLAN.md, STATE.md)
- `out/` — Rendered video output

## Skills Used

This project uses these Claude Code skills:
- `kie-ai-core` — API auth, task polling, credits
- `kie-ai-image` — Image generation (4o Image, Flux Kontext, etc.)
- `kie-ai-video` — Video generation (Veo 3.1, Runway, etc.)
- `elevenlabs` — TTS, sound effects, dialogue
- `remotion` — Video composition patterns

## Commands

- `npm start` — Open Remotion Studio (preview)
- `npm run render` — Render 16:9 video
- `npm run render:shorts` — Render 9:16 video
- `npm run render:square` — Render 1:1 video
